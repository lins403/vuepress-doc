# PDF

  • <iframe>、<embed>、<object>

  • GitHub - mozilla/pdf.js: PDF Reader in JavaScript (opens new window)

    • 一个完整的 PDF 查看器，可以直接使用其提供的 viewer.html 查看 PDF 内容，包含完整样式和相关功能

  • GitHub - mozilla/pdfjs-dist: Generic build of PDF.js library. (opens new window)

    • 只包含 PDF 内容的渲染功能，需要借助canvas来渲染，同时需要自定义样式和功能

# 在vue项目中引入

将下载的文件解压重命名为pdf.js，放置在 public/static 下

tree -L 3 public
.
├── favicon.ico
├── index.html
└── static
    ├── demo.pdf
    └── pdf.js
        ├── LICENSE
        ├── build
        └── web

# 使用

// 不能跨域
const pdfUrl = '/static/demo.pdf'
window.open(`static/pdf.js/web/viewer.html?file=${pdfUrl}`)

# 支持跨域

1）配置CORS或者代理

// vue.config.js
module.exports = {
  devServer: {
    host: '0.0.0.0',
    port: 1024,
    proxy: {
      '/remote-assets': {
        target: 'https://www.*******',
        pathRewrite: { '^/remote-assets': '' }
      }
    }
  }
}

const pdfUrl = 'http://localhost:1024/remote-assets/pdf/demo.pdf'
window.open(`static/pdf.js/web/viewer.html?file=${pdfUrl}`)

2）将这段异常抛出代码注释掉

// public/static/pdf.js/web/viewer.js
if (origin !== viewerOrigin && protocol !== "blob:") {
  throw new Error("file origin does not match viewer's");
}
