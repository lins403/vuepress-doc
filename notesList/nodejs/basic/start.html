# 全局变量

global 全局变量 (opens new window)

  • __dirname：当前模块的路径名
  • __filename：当前模块的文件名
  • exports：对 module.exports 的引用
  • module：对当前模块的引用
  • require()：用于导入模块、JSON 和本地文件
    • CommonJS特点 > 文件加载机制 (opens new window)
