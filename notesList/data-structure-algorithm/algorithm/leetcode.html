# 二叉堆

二叉堆在逻辑上其实是一种特殊的二叉树（完全二叉树），只不过存储在数组里，我们把数组索引作为指针。

二叉堆还分为最大堆和最小堆。

  • 最大堆的性质是：每个节点都大于等于它的两个子节点。
  • 最小堆的性质是：每个节点都小于等于它的子节点。

应用：实现优先级队列（插入或者删除元素的时候，元素会自动排序）
