# 在线代码编辑器

For Vue 3 core reproductions, try reproducing it in The SFC Playground (opens new window).

If it is a Vue 2 bug or cannot be reproduced in the playground, you can use CodePen (opens new window), CodeSandbox (opens new window) or provide a GitHub repo.

目前简易demo用CodePen，project用CodeSandbox，js的调试一般在本地
