# 后端架构

## java + SpringCloud

基于 SpringCloud 的微服务架构。整个项目分为了五个服务：

- 注册中心 `eureka_server`
- 用户服务 `user_service`
- 项目服务 `project_service`
- 评论服务 `comments_service`
- 服务网关 `api_gateway`
