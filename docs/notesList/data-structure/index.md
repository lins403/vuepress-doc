# 数据结构与算法

## 复杂度

|                  | 名称   | 说明                                                         |
| ---------------- | ------ | ------------------------------------------------------------ |
| O(1)             | 常量   | 无论多少值，执行时间都不变。表示简单值和保存在变量中的值     |
| O(logn)          | 对数   | 执行时间随着值的增加而增加，但算法完成不需要读取每个值。例子:二分查找 |
| O(n)             | 线性   | 执行时间与值的数量直接相关。例子:迭代数组的所有元素          |
| O(n<sup>2</sup>) | 二次方 | 执行时间随着值的增加而增加，而且每个值至少要读取 n 次。例子:插入排序 |
